# Image Skill

Навык для работы с изображениями через AI модели.

## Когда использовать

✅ **ИСПОЛЬЗОВАТЬ когда:**

- Нужно сгенерировать изображение по описанию
- Описать содержимое изображения
- Показать список сгенерированных изображений

## Инструменты

### image_generate
Сгенерировать изображение по текстовому описанию.
```json
{"prompt": "A cute cat playing in the garden", "size": "1024x1024"}
```

Размеры:
- `1024x1024` - квадратный
- `768x1344` - портретный
- `1344x768` - альбомный

### image_describe
Описать содержимое изображения.
```json
{"image_path": "cat.png"}
```

### image_list
Показать список сгенерированных изображений.
```json
{}
```
