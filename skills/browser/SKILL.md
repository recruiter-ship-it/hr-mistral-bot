# Browser Skill

Навык для работы с веб-ресурсами.

## Когда использовать

✅ **ИСПОЛЬЗОВАТЬ когда:**

- Нужно найти информацию в интернете
- Получить содержимое веб-страницы
- Извлечь ссылки со страницы
- Проверить доступность URL

## Когда НЕ использовать

❌ **НЕ ИСПОЛЬЗОВАТЬ когда:**

- Нужно выполнить команду → используй `terminal_execute`
- Нужно работать с файлами → используй `filesystem`
- Нужно работать с Google Docs → используй `google` навык

## Инструменты

### browser_search
Поиск в интернете.
```json
{"query": "latest AI news", "num_results": 10}
```

### browser_fetch
Получить содержимое веб-страницы.
```json
{"url": "https://example.com", "extract_text": true}
```

### browser_check_url
Проверить доступность URL.
```json
{"url": "https://example.com"}
```
