# Voice Skill

Навык для работы с голосом: транскрибация и генерация речи.

## Когда использовать

✅ **ИСПОЛЬЗОВАТЬ когда:**
- Пользователь прислал голосовое сообщение
- Нужно озвучить текст (TTS)
- Нужно конвертировать аудиофайл в текст

## Инструменты

### voice_transcribe
Транскрибировать голосовое сообщение в текст.
```json
{
  "file_path": "путь к аудиофайлу",
  "language": "ru"
}
```

### voice_generate
Сгенерировать речь из текста (Text-to-Speech).
```json
{
  "text": "Текст для озвучивания",
  "voice": "default",
  "speed": 1.0
}
```

### voice_list
Показать доступные голоса.
```json
{}
```

## Поддерживаемые форматы

- MP3
- WAV
- OGG (Telegram голосовые)
- M4A
- WebM
