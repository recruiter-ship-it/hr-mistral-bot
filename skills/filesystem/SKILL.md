# Filesystem Skill

Навык для работы с файловой системой в изолированном workspace.

## Когда использовать

✅ **ИСПОЛЬЗОВАТЬ когда:**

- Нужно прочитать содержимое файла
- Создать новый файл или директорию
- Записать данные в файл
- Найти файлы по имени или содержимому
- Получить информацию о файле

## Когда НЕ использовать

❌ **НЕ ИСПОЛЬЗОВАТЬ когда:**

- Нужно выполнить программу → используй `terminal_execute`
- Нужно получить веб-страницу → используй `browser_fetch`
- Нужно работать с Google Docs → используй `google` навык

## Инструменты

### fs_read_file
Прочитать содержимое файла.
```json
{"path": "notes.txt"}
```

### fs_write_file
Записать содержимое в файл.
```json
{"path": "output.md", "content": "# Hello\\nWorld!", "mode": "write"}
```

### fs_list_dir
Показать содержимое директории.
```json
{"path": "", "pattern": "*.md"}
```

### fs_search
Поиск файлов.
```json
{"query": "report", "search_type": "name"}
```
